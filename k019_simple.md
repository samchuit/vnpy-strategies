# K019 Simple 策略

## 📌 策略概述

| 项目 | 内容 |
|------|------|
| 策略名称 | K019 Simple |
| 策略类型 | 趋势跟踪 |
| 时间框架 | 日线 / 60分钟 |
| 交易品种 | 期货主力连续合约 |

## 🎯 策略逻辑

### 核心指标
1. **MA均线系统**
   - 短期均线: 5日 MA
   - 长期均线: 20日 MA
   - 金叉买入，死叉卖出

2. **OBV能量潮**
   - OBV 5日均线
   - OBV金叉确认趋势

3. **止损止盈**
   - 止损: 3%
   - 止盈: 8%
   - 追踪止损: 2%

### 买入条件
```
OBV > OBV_MA AND MA5 > MA20
```

### 卖出条件
```
收益 < -3% OR 收益 > 8%
```
